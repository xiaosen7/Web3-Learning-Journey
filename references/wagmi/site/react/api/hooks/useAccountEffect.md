# useAccountEffect

Hook for listening to account lifecycle events.

## Import

```ts
import { useAccountEffect } from 'wagmi'
```

## Usage

::: code-group
```tsx [index.tsx]
import { useAccountEffect } from 'wagmi'

function App() {
  useAccountEffect({
    onConnect(data) {
      console.log('Connected!', data)
    },
    onDisconnect() {
      console.log('Disconnected!')
    },
  })
}
```
