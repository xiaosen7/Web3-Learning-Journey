# useSignMessage

Composable for signing messages.

## Import

```ts
import { useSignMessage } from '@wagmi/vue'
```

## Usage

::: code-group
```vue [index.vue]
<script setup lang="ts">
import { useSignMessage } from '@wagmi/vue'

const { signMessage } = useSignMessage()
</script>

<template>
  <button @click="signMessage({ message: 'hello world' })">
    Sign message
  </button>
</template>
```

:::
